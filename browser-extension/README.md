# 拾迹 TraceNest 浏览器扩展

Chrome 与 Edge 共用的 Manifest V3 扩展，用于保存当前网页和选中文字。

## 本地安装

1. 解压 `TraceNest-Extension-v0.2.0.zip`。
2. Chrome 打开 `chrome://extensions`；Edge 打开 `edge://extensions`。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择解压后的文件夹。
5. 将“拾迹 TraceNest”固定到浏览器工具栏。

## 功能

- 工具栏弹窗保存当前网页。
- 选中文字后打开弹窗，直接保存摘录。
- 右键菜单快速保存网页或选中文字。
- 填写“为什么收藏”。
- 重复链接提示。
- 网络或登录失败时暂存最后一次内容。

首次使用前，请先在同一浏览器中打开拾迹网页版并完成访问验证。
